# 🚀 COMO PUBLICAR O SITE EMD SERVIÇOS (100% GRÁTIS)

## 📱 MÉTODO MAIS FÁCIL: VERCEL (Recomendado)

### ✅ Passo 1: Criar conta no GitHub (Gratuito)
1. Acesse: https://github.com/signup
2. Crie sua conta (use seu email)
3. Confirme o email

### ✅ Passo 2: Fazer Upload do Site para GitHub
1. Acesse: https://github.com/new
2. Nome do repositório: `emd-servicos-site`
3. Marque como **Público**
4. Clique em "Create repository"
5. Na página que abrir, clique em "uploading an existing file"
6. **Arraste TODOS os arquivos do projeto** (menos node_modules se tiver)
7. Clique em "Commit changes"

### ✅ Passo 3: Publicar com Vercel (Gratuito Forever)
1. Acesse: https://vercel.com/signup
2. Clique em "Continue with GitHub"
3. Autorize a conexão com GitHub
4. Clique em "Import Project"
5. Selecione o repositório `emd-servicos-site`
6. Clique em "Deploy"
7. **PRONTO!** Em 2 minutos seu site estará no ar! 🎉

### 📱 Passo 4: Seu Link e QR Code
Após o deploy, você receberá um link tipo:
```
https://emd-servicos-site.vercel.app
```

**GERAR QR CODE (GRATUITO):**
1. Copie o link do seu site
2. Acesse: https://www.qr-code-generator.com/
3. Cole o link
4. Clique em "Create QR Code"
5. Baixe a imagem PNG
6. **Pronto para imprimir ou enviar!** 📲
