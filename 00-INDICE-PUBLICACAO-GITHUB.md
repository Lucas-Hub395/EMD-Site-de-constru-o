## 🔍 BUSCA RÁPIDA

### Problemas Comuns

**"Module not found: figma:asset"**
→ `/IMAGENS-UNSPLASH-PRONTAS.md`
→ Você não substituiu as importações

**"Permission denied (publickey)"**
→ `/COMANDOS-GIT.md` → Seção "Autenticação"
→ Use Personal Access Token

**"Build failed"**
→ `/README-GITHUB.md` → Seção "Problemas"
→ Execute: `npm install && npm run build`

**"Site não indexa no Google"**
→ `/INSTRUCOES-PUBLICACAO-GITHUB.md` → Seção "Indexação"
→ Aguarde 2-7 dias, envie sitemap

**"Como fazer upload das fotos?"**
→ `/IMAGENS-UNSPLASH-PRONTAS.md` → Seção "Logo EMD"
→ Use Imgur ou /public/images/

**"Git: not a repository"**
→ `/COMANDOS-GIT.md` → Seção "Problemas Comuns"
→ Execute: `git init`

## 📚 GLOSSÁRIO

**Git** - Sistema de controle de versão
**GitHub** - Plataforma de hospedagem de código
**Netlify** - Plataforma de deploy/hospedagem
**Deploy** - Publicar o site online
**Build** - Compilar o código para produção
**Commit** - Salvar mudanças no Git
**Push** - Enviar código para o GitHub
**Repository (Repo)** - Projeto no GitHub
**Branch** - Ramificação do código
**Merge** - Mesclar códigos
**SEO** - Otimização para motores de busca
**Sitemap** - Mapa do site para Google
**Indexação** - Google adicionar site ao índice
**CDN** - Rede de distribuição de conteúdo
**SSL/HTTPS** - Certificado de segurança
**Unsplash** - Banco de imagens gratuitas
**Placeholder** - Imagem temporária
