# 🚀 PUBLICAR SITE AGORA - 3 PASSOS RÁPIDOS

## ⚡ OPÇÃO 1: VERCEL (Mais Rápido - 5 minutos)

### 1. Acesse: https://vercel.com
- Clique "Sign Up" → "Continue with GitHub"
- Autorize a Vercel

### 2. Fazer Deploy DIRETO (SEM GitHub)
- No dashboard, clique **"Add New..."** → **"Project"**
- Clique em **"Deploy without Git Provider"**
- **Arraste a PASTA INTEIRA** `emdservicos` para o upload
- **OU** clique para selecionar os arquivos:
  - Selecione TODOS os arquivos e pastas do projeto
  - NÃO incluir `node_modules` se existir
- Clique **"Deploy"**

### 3. Aguarde 2-3 minutos
- Vercel fará build e publicará
- Você receberá um link: `https://emdservicos.vercel.app`
- **PRONTO! Site no ar!**

### 4. Domínio Personalizado (Opcional - depois)
- Settings → Domains → Add `emdservicos.pt`
- Configure DNS conforme instruções

---

## ⚡ OPÇÃO 2: NETLIFY DROP (Arraste e Solte)

### 1. Acesse: https://app.netlify.com/drop

### 2. Preparar Pasta
- Abra terminal na pasta do projeto
- Execute: `npm install && npm run build`
- Será criada a pasta `dist`

### 3. Arraste a pasta `dist` para o Netlify Drop
- O site será publicado instantaneamente!
- Link: `https://[nome-aleatorio].netlify.app`

### 4. Domínio (Opcional - depois)
- Domain settings → Add custom domain → `emdservicos.pt`

---

## ⚡ OPÇÃO 3: GitHub + Vercel (Mais Completo - 15 minutos)

### 1. Criar Repositório GitHub
```bash
# No terminal, na pasta do projeto:
git init
git add .
git commit -m "EMD Serviços website"
git branch -M main
```

### 2. No GitHub
- Crie novo repositório: https://github.com/new
- Nome: `emdservicos`
- Copie a URL

### 3. Upload
```bash
git remote add origin https://github.com/[SEU-USUARIO]/emdservicos.git
git push -u origin main
```

### 4. Deploy Vercel
- https://vercel.com → "New Project"
- Importar do GitHub → Selecionar `emdservicos`
- Deploy (automático)
