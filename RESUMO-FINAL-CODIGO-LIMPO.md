## 🚀 COMO PUBLICAR AGORA

### Passo 1: Adicionar Imagens (Escolha A ou B)

**Opção A: Suas Fotos Reais** (Recomendado)
```bash
mkdir -p public/images
# Copie suas 18 fotos para esta pasta
```

**Opção B: URLs Temporárias** (Mais Rápido)
- Abra `IMAGENS-NECESSARIAS.md`
- Copie as URLs do Unsplash
- Cole nos 4 componentes mencionados acima

### Passo 2: Testar Localmente
```bash
npm install
npm run dev
```
✅ Verifique se tudo funciona sem erros

### Passo 3: Git + GitHub
```bash
git init
git add .
git commit -m "🎉 Site EMD Serviços - Pronto para produção"

# Crie o repositório em: https://github.com/new
# Nome: emdservicos

git remote add origin https://github.com/SEU-USUARIO/emdservicos.git
git branch -M main
git push -u origin main
```

### Passo 4: Deploy Netlify
1. https://app.netlify.com/
2. "Add new site" → "Import from Git"
3. Selecione: emdservicos
4. Build: `npm run build`
5. Publish: `dist`
6. Deploy!
