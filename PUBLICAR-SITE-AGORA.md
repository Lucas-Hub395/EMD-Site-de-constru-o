# 🚀 PUBLICAR SITE AGORA - SUPER SIMPLES

## ✅ MÉTODO MAIS RÁPIDO - NETLIFY DROP

### PASSO 1: Preparar os arquivos

No Figma Make, você já tem tudo pronto!

### PASSO 2: Ir para Netlify Drop

1. **Abra este link no navegador:**
   ```
   https://app.netlify.com/drop
   ```

2. **Crie uma conta rápida** (ou faça login):
   - Pode usar conta Google
   - Ou criar com email

### PASSO 3: Fazer o Deploy

**OPÇÃO A - Se você conseguir baixar a pasta:**
1. Baixe a pasta do projeto no seu Mac
2. Arraste a pasta para a área do Netlify Drop
3. Aguarde o upload (1-2 minutos)
4. **PRONTO! Site no ar!** 🎉

**OPÇÃO B - Via GitHub (recomendado):**

Se o Figma Make tem integração com GitHub:
1. Clique em "Push to GitHub" ou "Deploy to GitHub"
2. Crie um repositório: `emd-servicos-site`
3. No Netlify, clique em "Import from Git"
4. Conecte sua conta GitHub
5. Selecione o repositório
6. Clique em "Deploy"
7. **PRONTO! Site no ar!** 🎉

## 📍 DEPOIS DO DEPLOY

O Netlify vai te dar um link tipo:

```
https://seu-site-12345.netlify.app
```

**GUARDE ESTE LINK!** É o link do seu site no ar.

---

## 🔍 ADICIONAR NO GOOGLE SEARCH CONSOLE

### Com o link do site em mãos:

1. **Vá em:** https://search.google.com/search-console

2. **Adicionar propriedade:**
   - Clique em "Adicionar propriedade"
   - Cole o link: `https://seu-site-12345.netlify.app`
   - Clique em "Continuar"

3. **Verificar o site:**
   - Google vai pedir verificação
   - Escolha "Tag HTML" ou "Arquivo HTML"
   - Siga as instruções do Google

4. **Enviar Sitemap:**
   - Após verificar, vá em "Sitemaps"
   - Adicione: `https://seu-site-12345.netlify.app/sitemap.xml`
   - Clique em "Enviar"

5. **Solicitar Indexação:**
   - Vá em "Inspeção de URL"
   - Cole: `https://seu-site-12345.netlify.app`
   - Clique em "Solicitar indexação"

**Pronto!** Em 3-7 dias seu site aparece no Google!
