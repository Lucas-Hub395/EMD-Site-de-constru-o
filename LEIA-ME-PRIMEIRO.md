## 💻 COMANDOS ÚTEIS

### Desenvolvimento Local
```bash
# Instalar dependências
npm install

# Iniciar servidor desenvolvimento
npm run dev

# Abre em: http://localhost:3000
```

### Build de Produção
```bash
# Criar build otimizada
npm run build

# Preview da build
npm run preview
```

### Git/GitHub
```bash
# Inicializar Git
git init

# Adicionar arquivos
git add .

# Commit
git commit -m "🎉 Primeiro commit - Website EMD Serviços"

# Conectar GitHub
git remote add origin https://github.com/[SEU-USUARIO]/emdservicos.git

# Enviar código
git branch -M main
git push -u origin main
```

Para mais comandos, veja: **GIT_COMANDOS.md**
