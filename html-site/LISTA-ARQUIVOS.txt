   📄 Página principal do site

   🎨 Arquivo de estilos
   ⚙️ Contém: Todos os estilos, cores, animações, responsividade

   ⚡ Arquivo JavaScript
   ⚙️ Contém: Funcionalidades (menu mobile, carrossel, formulários)
   💡 Faz o site funcionar (interatividade)

   ⚙️ Contém: Regras de deploy, headers de segurança
   💡 Necessário para publicar no Netlify

5. robots.txt
   🤖 Arquivo para buscadores (Google, Bing)
   ⚙️ Contém: Permissões para indexação
   💡 Ajuda o site a aparecer no Google

6. sitemap.xml
   🗺️ Mapa do site
   ⚙️ Contém: Lista de todas as páginas/seções
   💡 Ajuda Google a entender estrutura do site

7. .gitignore
   📝 Arquivo do Git
   ⚙️ Contém: Lista de arquivos a ignorar no Git
   💡 Usado se publicar via GitHub
