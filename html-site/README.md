## 🚀 Como Publicar no Netlify

### Método 1: Arraste e Solte (Mais Rápido)

1. Acesse [app.netlify.com](https://app.netlify.com)
2. Faça login (ou crie conta grátis)
3. Arraste a pasta `html-site` para o Netlify
4. Pronto! Seu site estará no ar em segundos

### Método 2: Via GitHub (Recomendado)

#### Passo 1: Criar Repositório no GitHub

1. Acesse [github.com](https://github.com)
2. Clique em **"New repository"** (botão verde)
3. Nome do repositório: `emd-servicos-site`
4. Descrição: `Site oficial da EMD Serviços`
5. Selecione **Public**
6. Clique em **"Create repository"**

#### Passo 2: Upload dos Arquivos

**Opção A - Pelo Site do GitHub (Mais Fácil)**

1. No seu novo repositório, clique em **"uploading an existing file"**
2. Arraste todos os arquivos da pasta `html-site`
3. Escreva no commit: `Adicionar site EMD Serviços`
4. Clique em **"Commit changes"**

**Opção B - Via Git (Linha de Comando)**

```bash
# 1. Abra o terminal na pasta html-site
cd html-site

# 2. Inicialize o repositório Git
git init

# 3. Adicione todos os arquivos
git add .

# 4. Faça o primeiro commit
git commit -m "Adicionar site EMD Serviços"

# 5. Conecte ao GitHub (substitua SEU-USUARIO)
git branch -M main
git remote add origin https://github.com/SEU-USUARIO/emd-servicos-site.git

# 6. Envie os arquivos
git push -u origin main
```

#### Passo 3: Conectar ao Netlify

1. Acesse [app.netlify.com](https://app.netlify.com)
2. Clique em **"Add new site"** → **"Import an existing project"**
3. Escolha **"GitHub"**
4. Selecione o repositório `emd-servicos-site`
5. Configurações:
   - **Build command**: deixe vazio
   - **Publish directory**: deixe vazio ou coloque `.`
6. Clique em **"Deploy site"**

#### Passo 4: Configurar Domínio Personalizado (Opcional)

1. No painel do Netlify, vá em **"Domain settings"**
2. Clique em **"Add custom domain"**
3. Digite: `emdservicos.pt` (ou outro domínio)
4. Siga as instruções para configurar DNS
